Detailed Description

Second set of exercises.

These exercises consist of an introduction to process creation.

The following system calls and functions are covered:

  • mmap
  • munmap
  • pipe
  • fork
  • wait
  • waitpid
  • kill
  • raise
  • read
  • write
  • close
  • execvp
  • system
