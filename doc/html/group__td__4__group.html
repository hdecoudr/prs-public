Detailed Description

Fourth set of exercises.

These exercises consist of an introduction to signals and stack manipulation.

The following system calls and functions are covered:

  • sigprocmask
  • sigaction
  • sigsuspend
  • fork
  • waitpid
  • kill
  • pause
  • execvp
  • sigemptyset
  • sigaddset
  • setjmp
  • longjmp
