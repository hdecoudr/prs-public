Detailed Description

Third set of exercises.

These exercises consist of an introduction to communication between processes.

The following system calls and functions are covered:

  • pipe
  • pipe2
  • fork
  • wait
  • open
  • dup2
  • lseek
  • read
  • write
  • close
  • execl
  • execlp
  • execvp
