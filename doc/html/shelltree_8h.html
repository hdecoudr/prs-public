◆ yyparse()

int yyparse ( void  )

The yyparse() function is used by the Bison parser.

This function reads tokens, executes actions, and ultimately returns when it encounters end-of-input or an unrecoverable syntax error. You can also write an action which directs yyparse to return immediately without reading further.

Returns
This function can return the following values:
  • YYACCEPT if parsing was successful (return is due to end-of-input)
  • YYABORT if parsing failed because of invalid input, i.e., input that contains a syntax error or that causes YYABORT to be invoked
  • YYNOMEM if parsing failed due to memory exhaustion
