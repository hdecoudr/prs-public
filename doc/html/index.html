About

The following documentation covers system programming exercises under Linux of the PRS teaching module.

These exercises consist of an introduction to system programming under Linux and address the following concepts:

  • Inputs and outputs;
  • Process creation;
  • Process communication;
  • Signals manipulation;
  • Threads manipulation;
  • Parallelism.

All exercises are documented and are available either from the Files menu or the Modules menu.

The exercises are grouped by theme and organized as follows:

Exercises Description
td-1 Introductory exercises to inputs and outputs using system calls.
td-2 Introductory exercises to processes creation using system calls.
td-3 Introductory exercices to signals and stack manipulation using system calls.
td-4 Introductory exercices to signals and stack manipulation using system calls.
td-5 Introductory exercises to threads using pthread.
td-6 Introductory exercises to OpenMP.

Prerequisites

Install the following prerequisites in order to build the exexercises:

  • GNU GCC;
  • GNU Make;
  • GNU GDB.
sudo apt-get install gcc make gdb

Additionally, if you want to build the shelltree exercise of the td-2 module, install the following prerequisites:

  • Flex;
  • GNU Bison;
  • GNU Readline.
sudo apt-get install flex bison libreadline-dev

Finally, install the following prerequisites if you want to modify this documentation:

  • Doxygen
  • Graphviz
sudo apt-get install doxygen graphviz

Build the exercises

A Makefile file is available within each directory corresponding to a set of exercises. Navigate to the desired td-* directory and invoke make to build all the exercises of the directory:

cd <td_number> && make

A bin and an obj directory will be generated. The bin directory contains all the executables while the obj directory contains object files.

To a source file corresponds an executable with the same name (except for nested directories).

Execute an exercise

Nagivate to the generated bin directory of a set of exercises and execute the desired exercise:

cd bin && ./<exercise>
