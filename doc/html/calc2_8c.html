Detailed Description

Exercise 3.6.

Distributes by block a calculation through several processes.

This program uses the following system calls:

  • fork(void)
  • wait(int* wstatus)
  • open(const char* pathname, int flags, mode_t mode)
  • lseek(int fd, off_t offset, int whence)
  • write(int fd, const void* buf, size_t count)
  • close(int fd)

◆ exit_on_error()

void exit_on_error ( int  assertion)
static

The exit_on_error() function exits the program if the assertion parameter is evaluated to TRUE.

If the assertion is evaluated to TRUE and errno is set, then the error number and its associated message are displayed. Otherwise, a generic message is displayed.
