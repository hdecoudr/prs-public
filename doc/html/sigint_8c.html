static void exit_on_error ( int  assertion)
static

The exit_on_error() function exits the program if the assertion parameter is evaluated to TRUE.

If the assertion is evaluated to TRUE and errno is set, then the error number and its associated message are displayed. Otherwise, a generic message is displayed.
