Detailed Description

Exercise 3.2.

Executes a two commands by using a pipe.

This program uses the following system calls and functions:

  • pipe(int pipefd[2])
  • fork(void)
  • wait(int* wstatus)
  • dup2(int oldfd, int newfd)
  • close(int fd)
  • execlp(const char* file, const char* arg, ..., NULL)
  • execvp(const char* file, char* const argv[])

◆ exit_on_error()

void exit_on_error ( int  assertion)
static

The exit_on_error() function exits the program if the assertion parameter is evaluated to TRUE.

If the assertion is evaluated to TRUE and errno is set, then the error number and its associated message are displayed. Otherwise, a generic message is displayed.
